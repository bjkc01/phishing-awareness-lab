# Employee Awareness Guide

This guide is designed to help employees understand phishing risks and how to respond safely.

## Recognize phishing indicators

- unexpected sender or domain
- urgent requests for action
- unfamiliar links or attachments
- requests for credentials or sensitive data

## Verification steps

1. Hover over links before clicking.
2. Confirm sender identity through a second channel.
3. Do not enter credentials on untrusted pages.
4. Report suspicious messages immediately.

## Reporting process

- Use the designated phishing report button.
- Forward suspicious emails to the security team.
- Do not reply to or interact with the suspicious sender.

## Best practices

- use strong, unique passwords
- enable multi-factor authentication where available
- keep software and browsers updated
- complete security awareness training on schedule
