# Basic Remediation Training

This training module is intended for employees who demonstrated low-risk behavior in the phishing simulation.

## Learning objectives

- understand phishing techniques
- recognize suspicious email elements
- review secure browsing habits
- learn reporting expectations

## Training topics

1. Phishing fundamentals
2. Social engineering examples
3. Best practices for email handling
4. When and how to report suspicious content

## Recommended action

- complete the employee awareness guide
- review company security policies
- attend the next scheduled awareness session
- practice identifying simulated phishing indicators
