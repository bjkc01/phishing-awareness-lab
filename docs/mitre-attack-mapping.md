# MITRE ATT&CK Mapping

This lab maps directly to the MITRE ATT&CK technique:

- **T1566 – Phishing**

## Technique details

- **T1566.001 – Spearphishing Attachment**: phishing messages with attachments.
- **T1566.002 – Spearphishing Link**: phishing messages with malicious links.
- **T1566.003 – Spearphishing via Service**: phishing delivered through web or cloud services.

## Project relevance

`phishing-awareness-lab` focuses on phishing simulation and user response tracking. It emphasizes:

- identifying successful social engineering attempts,
- measuring click-through behavior,
- evaluating credential submission risk,
- testing reporting and escalation paths.

## Defensive alignment

Using the MITRE framework in a training lab demonstrates the ability to connect security exercises with enterprise threat models. This is valuable for:

- SOC analyst workflows,
- incident response communication,
- maturity assessments,
- security awareness program documentation.
